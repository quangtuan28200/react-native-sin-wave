# react-native-sin-wave

A React Native library for generating and playing sine wave sounds.

## Installation

```sh
npm install react-native-sin-wave
```

## Usage


```js
import { multiply } from 'react-native-sin-wave';

// ...

const result = await multiply(3, 7);
```
